#  GESTOR+ 

---

## 🧾 Sistema de Gerenciamento de Estoque

Sistema desenvolvido em Python para gerenciamento de produtos com persistência de dados em MySQL. O projeto implementa operações CRUD (Create, Read, Update e Delete), validação de dados de entrada e estrutura modular para organização da lógica de negócio e acesso ao banco de dados.

## 👁‍🗨 Visão Geral

O objetivo do projeto é fornecer uma aplicação de linha de comando (CLI) capaz de gerenciar produtos armazenados em banco de dados relacional, aplicando conceitos fundamentais de desenvolvimento back-end, modelagem de dados e integração com banco de dados.
